# pymobile-mcp

Pure Python MCP server for mobile automation — Android via uiautomator2, iOS via pymobiledevice3.

## Quick Start

```bash
pip install pymobile-mcp
```

```json
{
  "mcpServers": {
    "pymobile-mcp": {
      "command": "pymobile-mcp",
      "args": ["run"]
    }
  }
}
```
